Video Tag

Mit folgendem Statement, könnte mann ein Film oder ein Video in eine Webseite einbetten. Dabei muss beachtet werden, dass YouTube videos am besten mit iFrames eingebettet werden sollten, da es sonst zu komplikationen kommen kann:

<video width="200px" height="150px" controls>
<source src="film.mp4" type="video/mp4">
</video>

In diesem Beispiel, wird ein Video im Mp4 format eigebettet. Es hat den Namen "Film", und der user kann auf die video kontrollfunktionen zugreiffen. Das video hat eine breite von 200px und eine höhe von 150px.

CSS Variablen

CSS Variabeln können verwendet werden, um Werte zentral zu speichern. Dies hilft dabei, wenn man z.B. die Farben mehrmals ändert. Durch die Variabeln, muss man nicht bei allen Regeln einzeln den Wert ändern, sondern nur bei der definition der Variable. Hier ein beispiel, wie mann eine CSS Variable definiert:

:root{
--farbe : white;
}

Und hier wie sie angewendet wird:

p{
color: var(--farbe);
}

Die Variable wird also mit "var()" eingeleitet, und zwischen die Klammern geschrieben.
